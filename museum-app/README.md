# Музей БелОГ — ПО для тач-экранов

Интерактивное программное обеспечение для музея Белорусского общества глухих.

## Возможности

- **Киоск** для горизонтального (`952 × 535 мм`) и вертикального (`616 × 1096 мм`) тач-экранов
- **Главная страница** с интерактивными элементами: флаг, герб, логотип
- **Эффект перелистывания** страниц как в книге + свайпы
- **Сетка разделов** с карточками
- **Админ-панель** — 10 слотов разделов со слайдовым редактором (как PowerPoint), оформление сайта, rich-text
- **Слайды разделов** — стартовые шаблоны + свободное размещение текста, медиа и фигур; на киоске листание свайпом
- **Единый фон** проекта и letterbox-масштаб под экраны 952×535 и 616×1096 мм

## Быстрый старт

```bash
cd museum-app
npm install
cp .env.local.example .env.local
npm run dev
```

Откройте:
- Главная: http://localhost:3000
- Горизонтальный киоск: http://localhost:3000/display/horizontal
- Вертикальный киоск: http://localhost:3000/display/vertical
- Админ-панель: http://localhost:3000/admin/site (разделы — в боковом меню, слоты 01–10)

## Вход в админ-панель

| | |
|---|---|
| **Логин** | `belog_admin` |
| **Пароль** | `MuzeiBelOG2026` |

Пароль можно изменить в файле `.env.local`.

## Настройка киоска в музее

1. Запустите сервер: `npm run build && npm start`
2. Откройте Chrome/Edge в полноэкранном режиме на соответствующем тач-экране:
   - Горизонтальный (952×535 мм): `http://<IP-сервера>:3000/display/horizontal`
   - Вертикальный (616×1096 мм): `http://<IP-сервера>:3000/display/vertical`
3. Рекомендуется отключить спящий режим Windows на ПК киоска

## Медиа-загрузка

Во всех полях загрузки в админке поддерживаются:

| Тип | Форматы | Лимит |
|-----|---------|-------|
| Изображение | JPEG, PNG, WebP, GIF | 10 МБ |
| Видео | MP4, WebM | 200 МБ |
| Аудио | MP3, WAV, OGG | 30 МБ |
| PDF | PDF | 25 МБ |
| Презентация | PPTX | 80 МБ |

Файлы сохраняются в `public/uploads/{тип}/` и **не коммитятся** в git.

На киоске все типы открываются во встроенном просмотрщике — без новых вкладок и внешних сервисов.

### PPTX (презентации)

Для конвертации слайдов при загрузке `.pptx` на сервере нужен **LibreOffice**:

```bash
# macOS
brew install --cask libreoffice

# Ubuntu/Debian
sudo apt install libreoffice

# Windows — установите LibreOffice с https://www.libreoffice.org
```

Конвертация: PPTX → PDF → PNG (по одному на слайд). Если LibreOffice недоступен, используется упрощённый fallback (качество ниже).

Рекомендуется зарезервировать место на диске для видео и презентаций (несколько ГБ).

## Структура

- `public/assets/` — фоны, флаг, гербы
- `public/uploads/` — загруженные медиа из админки (изображения, видео, PDF, аудио, слайды PPTX)
- `data/museum.db` — база данных SQLite (создаётся автоматически)

## Технологии

- Next.js 16, React, TypeScript
- SQLite + Drizzle ORM
- TipTap (rich-text редактор)
- Framer Motion (анимации)
- Tailwind CSS
